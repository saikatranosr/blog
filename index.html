How to istall VS Code on Android Phone - ProgrammerSaikat

Before going to these process make sure you have an External keyboard to type, you can't type on your virtual keyboard on Code Server
If you haven't one, dont worry... Click here for another tutorial
  1. Install Termux

    2. Go to Play Store and Install Termux

  2. apt update && apt upgrade

    3. Run the command in Termux to update packages:

    apt update && apt upgrade

    If it dosen't work, try to change repo by firing this command:

    termux-chnge-repo

    And then try update commands

  3. Install Ubuntu on Termux

    4. Fire the bellow command on Termux to install Ubuntu on Termux

    pkg install wget openssl-tool proot -y && hash -r && wget https://raw.githubusercontent.com/EXALAB/AnLinux-Resources/master/Scripts/Installer/Ubuntu/ubuntu.sh && bash ubuntu.sh
  4. Start Ubuntu

    5. Start Ubuntu on Termux by firing this comand:

    ./start-ubuntu.sh
  5. Install code server:

    6. First download the latest release of code server from Github by firing this command:

    wget https://github.com/cdr/code-server/releases/download/v3.10.2/code-server-3.10.2-linux-arm64.tar.gz

    After downloading code swerver you have to install code server. To intall, run this command:

    tar -xvf ./code-server-3.10.2-linux-arm64.tar.gz
  6. Start code-server

    7. Go to code-server-3.10.2-linux-arm64/bin folderin Termux

    cd code-server-3.10.2-linux-arm64/bin
    Note: You don't have to type this command manually or copy it everytime. Simply type cd code and Tab to auto comple this. afterthat, add /bin to this or go to bin folder manually cd bin

    Set a password by running this command:

    Note: Every time you open code-server you have to set a password.
    export PASSWORD="s"

    Then start code-server by running this command:

    ./code-server

    Congratulations!! Now you have successfully installed code-server on your mobile

    Go to browser and go to: 

    localhost:8080

    Enter your password and get started with code-server(Vs Code)
